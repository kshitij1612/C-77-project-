Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux. It comes with built-in support for JavaScript, TypeScript and Node.

In javascript, the console is an object which provides access to the browser debugging console. We can open a console in web browser by using: Ctrl + Shift + I for windows and Command + Option + I for Mac. The console object provides us with several different methods, like : log() error() warn() clear() time() and timeEnd() table() count() group() and groupEnd() custom console logs
